## 📝 Study Rule ##
- 기본 난이도는 프로그래머스 Lv.2 기준으로 정함
- 매주 오프라인으로 만나기 전까지 3문제 풀이 완료하기
- 매주 모임 시간에 발표 순서 정해서 코드 설명 및 질의응답

## 📝 Commit Rule ##
* 폴더 구조는 다음과 같다.
    * 주차별 폴더 - 문제 폴더 - 문제 소스코드 파일
        * 예) ```1주차```폴더 -> ```문제이름``` 폴더 -> ```hyunhee.cpp``` 
* 소스코드 파일은 누구 파일인지 알아볼 수 있게 작성
* 커밋 내용은 자유롭게
* 문제 ```4```는 보너스 문제
